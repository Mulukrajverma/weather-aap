# Weather App

This is a simple weather application built using React and Vite.
The app allows users to search for a city and view current weather information.
The main focus of this project was to practice API integration and component-based UI in React.

---

## What this app does
- Search weather by city name
- Shows temperature and weather condition
- Displays additional info like humidity and wind
- Uses a clean and simple UI
- Handles basic error cases (invalid city, empty search)

---

## Tech used
- React.js
- Vite
- JavaScript
- HTML & CSS
- Weather API
